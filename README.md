# Communication Engineering Project

A complete Python simulation of a communication system — from analog
AM/FM all the way to a live QPSK link over TCP sockets. Everything is
implemented from first principles using NumPy/SciPy, no black-box SDR
libraries.

---

## Project layout

```
src/
  analog.py     AM, DSB-SC, SSB, FM modulators & demodulators
  digital.py    BPSK, QPSK, M-QAM, RRC pulse shaping, matched filter
  channel.py    AWGN, multipath, Rayleigh, Rician, freq-offset, phase noise
  coding.py     Hamming(7,4), repetition code, BER/SER metrics, theory curves
  link.py       Full TX/RX pipeline + TCP socket transport (CLI included)

examples/
  ex1_analog_modulation.py     Waveforms, spectra, and recovered messages
  ex2_ber_curves.py            BER vs Eb/N0: sim vs theory
  ex3_constellation_animation.py   Animated constellation SNR sweep (GIF)
  ex4_channels.py              Multipath / Rayleigh / Rician effects
  ex5_end_to_end.py            Text TX with Hamming coding gain demo
  ex6_socket_loopback.py       Live TX↔RX over TCP loopback

outputs/        PNG plots generated by examples
animations/     GIF animations
```

## Live socket link (two terminals)

```bash
# Terminal 1 — receiver
python -m src.link receive --port 9000

# Terminal 2 — transmitter
python -m src.link transmit --port 9000 --message "Hello Munich" --snr 10
```

Omit `--snr` for a noiseless link. Add `--no-coding` to disable Hamming FEC.

## Module reference

### `src/analog.py`

| Function | What it does |
|----------|-------------|
| `am_modulate(m, fc, fs, mu)` | AM with carrier: `Ac[1 + mu*m(t)]cos(wc*t)` |
| `am_demodulate_envelope(s, fs, cutoff)` | Envelope detector via Hilbert |
| `dsb_sc_modulate / demodulate` | Double-sideband suppressed carrier |
| `ssb_modulate(m, fc, fs, sideband)` | Single sideband via phasing method |
| `fm_modulate(m, fc, fs, kf)` | FM via cumulative phase integral |
| `fm_demodulate(s, fs, kf)` | Instantaneous frequency via analytic signal |

**Key formula:**
```
FM:  s(t) = Ac · cos( 2π·fc·t + 2π·kf · ∫m(τ)dτ )
AM:  s(t) = Ac · [1 + μ·m(t)] · cos(2π·fc·t)
SSB: s(t) = m(t)cos(wc·t) ∓ m̂(t)sin(wc·t)   (∓ for USB/LSB)
```

### `src/digital.py`

| Function | Description |
|----------|-------------|
| `bpsk_modulate/demodulate` | 1 bit/symbol, {-1,+1} |
| `qpsk_modulate/demodulate` | 2 bits/symbol, Gray-coded |
| `qam_modulate/demodulate(bits, M)` | M ∈ {4,16,64,256}, square Gray-coded |
| `rrc_filter(beta, span, sps)` | Root-Raised-Cosine pulse |
| `pulse_shape(symbols, beta, span, sps)` | Upsample + RRC filter |
| `matched_filter(rx, h, sps)` | RRC matched filter + downsample |

**What RRC does:** The pulse-shaping filter controls spectral occupancy.
Using RRC at both TX (shape) and RX (matched filter) gives raised-cosine
overall response — zero ISI at the sampling instants.

---

### `src/channel.py`

| Function | Description |
|----------|-------------|
| `awgn(signal, snr_db)` | White Gaussian noise, SNR-accurate to ±0.05 dB |
| `multipath_channel(signal, taps)` | Tapped-delay-line: `h = [(delay_samples, gain), ...]` |
| `rayleigh_fading(signal, coherence_samples)` | Complex-Gaussian envelope |
| `rician_fading(signal, K_db, ...)` | Rician K-factor: K→∞ = AWGN, K→-∞ = Rayleigh |
| `frequency_offset(signal, fs, f_off)` | Carrier offset impairment |
| `phase_noise(signal, std_rad)` | Wiener-process phase drift |

---

### `src/coding.py`

| Function | Description |
|----------|-------------|
| `hamming74_encode(bits)` | Systematic Hamming(7,4): 4 data → 7 codeword bits |
| `hamming74_decode(bits)` | Syndrome decode: corrects 1 error per 7-bit block |
| `ber_bpsk_theory(ebn0_db)` | `Q(√(2·Eb/N0))` |
| `ber_qam_theory(ebn0_db, M)` | Approx BER for square M-QAM with Gray coding |

**Hamming rate = 4/7 ≈ 0.57.** The overhead is ~43% extra bandwidth in
exchange for single-error correction per codeword.

---

### `src/link.py`

Full transmit pipeline:
```
text → bits → Hamming(7,4) → QPSK → RRC pulse → AWGN → TCP socket
```

Full receive pipeline:
```
TCP socket → matched filter (RRC) → sample → QPSK demod → Hamming decode → text
```

Wire format: 8-byte header `[n_samples uint32, text_len uint32]` followed
by `n_samples × 8` bytes of interleaved float32 I/Q pairs.

## What each example demonstrates

| Example | Key lesson |
|---------|-----------|
| **ex1** | AM spectrum has carrier + two sidebands; SSB halves bandwidth; FM width follows Carson's rule: `B ≈ 2(Δf + fm)` |
| **ex2** | BPSK and QPSK share the same BER curve (same Euclidean distance per bit); 64-QAM needs ~8 dB more SNR than BPSK for equal BER |
| **ex3** | *(animation)* At low SNR constellation points blur into a cloud; at high SNR they collapse to tight clusters around ideal points |
| **ex4** | Multipath causes ISI → constellation clouds even at high SNR; Rayleigh fading BER falls only 10 dB/decade (vs. exponential for AWGN) |
| **ex5** | Hamming coding pushes the "error-free" threshold ~4 dB lower; text is readable at -2 dB coded vs. barely at 0 dB uncoded |
| **ex6** | Real TCP sockets: TX serialises IQ samples, RX deserialises and demodulates — proves the full pipeline works across process boundaries |

## Extending the project

Good next steps:

1. **OFDM** — add a cyclic prefix and an IFFT/FFT pair to turn
   frequency-selective multipath into flat per-subcarrier fading.
2. **LDPC / Turbo codes** — modern FEC with performance near the Shannon
   limit (≈ 1 dB from capacity).
3. **Phase-locked loop (PLL)** — carrier and timing recovery for a
   blind receiver that doesn't assume perfect synchronisation.
4. **GNU Radio integration** — replace the socket with a GR source/sink
   block to drive real SDR hardware (RTL-SDR, USRP).
5. **MIMO** — extend the channel to a matrix `H` and add
   zero-forcing or MMSE equalisers.
